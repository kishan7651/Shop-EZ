Build a COMPLETE PRODUCTION-READY FULL STACK E-COMMERCE WEB APPLICATION called **"ShopEZ"** similar to **Amazon / Flipkart** with modern UI, scalable architecture, secure authentication, payment gateway integration, and admin management.

TECH STACK

Frontend:

* React.js (Vite)
* Tailwind CSS
* React Router
* Axios
* Redux Toolkit (state management)
* Framer Motion (animations)
* React Icons
* React Query (optional for API caching)

Backend:

* Node.js
* Express.js

Database:

* MongoDB (Mongoose)

Authentication:

* JWT authentication
* bcrypt password hashing

Payments:

* Stripe or Razorpay payment gateway
* Support for UPI, card payments, and Cash on Delivery

Deployment:

* Frontend: Vercel / Netlify
* Backend: Render / Railway
* Database: MongoDB Atlas

The UI should look modern and professional similar to large e-commerce platforms.

The application must be **fully responsive (mobile, tablet, laptop, desktop)**.
